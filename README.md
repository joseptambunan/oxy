<p>Required Technology<p>

<p>Latest stable Bootstrap</p>
<p>Latest stable Laravel </p>

<p>Requirements</p>

<ul>
<li>Use basic Bootstrap to create simple Admin Control Panel.</li>
<li>Create Users Management with Admin and Non-Admin User Level.</li>
<li>Create Books Management.</li>
<li>Create Authors Management.</li>
<li>Create Login, Logout and Password Reset function.</li>
<li>Authors and Books should have a many-to-many database table relationship.</li>
<li>When creating or editing a Book, User should be able to select more than one Authors. </li>
<li>Try to use https://select2.org/ or https://selectize.github.io/selectize.js/. </li>
<li>Each Model landing page should have a table with list of record and action buttons.</li>
<li>Create at least ten dummy records for each Model.</li>
<li>Only Admin level can delete records.</li>
<li>Create a new Git and README.md using this document before starting project.</li>
<li>Push completed script to Git including database migration.</li>
</ul>

	<li>Install Laravel</li>
	<li>Set config db in .env</li>
	<li>Set config email in .env</li>
</ul>
